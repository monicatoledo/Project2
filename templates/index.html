Movie Data Analysis



The purpose of this project was to analyze :

  • How many movies were shown in the last ten years in some countries
  • What's the popularity of those movies
  • How much profits did those movies generate

    • Tools :

  • The Movie Database API (https://developers.themoviedb.org/3/movies/)
  • Python, Flask and DB
  • JavaScript, Leaflet, d3

    • Results :

  • A map with markers that represent movie numbers in each year in some countries
  • A map with markers that represent movie popularity
  • A time-sequence chart that represent profits by country and genre
